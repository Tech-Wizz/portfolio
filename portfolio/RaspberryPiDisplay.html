Raspberry Pi Display Project

Overview

This project uses a lightweight Raspberry Pi Zero W as a dedicated display device for an internal company dashboard. The goal was to create a reliable, low‑power unit capable of automatically launching an intraweb page on boot, running in fullscreen, and refreshing at regular intervals with minimal maintenance required.

Hardware

For this project, I used the CanaKit Raspberry Pi Zero W (Wireless) Complete Starter MAX Kit – 64GB Edition. This kit provided everything needed for a compact display computer:

  • Raspberry Pi Zero W (Wi‑Fi enabled)
  • Premium black case
  • 64GB microSD (pre‑loaded with NOOBS)
  • Power supply and HDMI adapter components

Since the only required function was opening and displaying a single intraweb page, the Pi Zero W's modest processing power was more than sufficient.

Operating System & Boot Customization

The system runs Raspberry Pi OS with customized boot graphics and icons to match company branding. Upon powering on, the Pi automatically launches a startup script (run.sh) which triggers the Python display program.

Automation & Display Logic

A Python script (tv.py) handles opening the dashboard page, switching to fullscreen mode, and refreshing the display at set intervals. The script uses:

  • webbrowser – launch the internal dashboard
  • pyautogui – simulate keyboard input (F11 fullscreen, F5 refresh)
  • xdotool – ensure the browser window is active

Python Code

import pyautogui
import webbrowser
import time
import os

pyautogui.FAILSAFE = False

url = "https://intraweb.company.com/dist/Shipping"

webbrowser.open(url)

time.sleep(60)

os.system('xdotool search --onlyvisible --class midori windowactivate')

pyautogui.press('f11')

# Get screen size
screen_width, screen_height = pyautogui.size()
print(screen_width)
print(screen_height)

# Move mouse to rest
pyautogui.moveTo(screen_width, screen_height)

while True:
    # Wait for 60 minutes (10=600 seconds)
    time.sleep(3600)
    # Execute the tab-reload command to refresh the page
    pyautogui.press('f5')
    print("Reloaded")
