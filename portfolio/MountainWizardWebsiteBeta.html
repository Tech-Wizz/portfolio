Mountain Wizard Website

Overview

The Mountain Wizard Website was created as the official online presence for my small business, Mountain Wizard LLC. Its purpose was to showcase the woodworking and handcrafted items I produced, while giving customers a simple, clean platform to view offerings, learn about the company, and get in touch.

The site was fully hosted on GitHub Pages, making it cost‑effective, version‑controlled, and easy to update. Product sales were handled through Etsy, with website links directing customers to the appropriate product listings for checkout.

Technologies Used

  • GitHub Pages – Static hosting for the entire website
  • HTML / CSS / JavaScript – Front‑end layout and structure
  • Etsy Storefront Integration – Redirects for product purchasing
  • Git Version Control – All source code publicly maintained on GitHub

Purpose and Features

The website functioned as the central marketing and informational hub for the business. It included:

  • Product Display Pages – Images, descriptions, and details about handmade items
  • Etsy Purchase Links – Each product page linked directly to Etsy for secure checkout
  • Business Contact Information – Email, social media, and business details
  • About the Company – Background story, mission, and craftsmanship philosophy
